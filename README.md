# NutriTrack
O NutriTrack é um aplicativo móvel destinado ao monitoramento alimentar e ao acompanhamento de objetivos nutricionais. Ele possibilita que o usuário lance os alimentos ingeridos durante o dia, verifique a quantidade de calorias consumidas e acompanhe o avanço em relação às metas diárias.
